# PlanetAPI
API básico usando node.js y express.js para formación

##Instalación
```
$ git clone https://github.com/yagoff/PlanetAPI.git
$ cd PlanetAPI
$ npm install
```
#Ejecución
`node server`

#Pruebas
`npm test`
